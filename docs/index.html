pwnDroid 0.1a

...all the tools you've been looking for

pwnDroid is a massive Android hacking and security collection that gives you direct access to rare and hard to find APKs with a short tutorial for each tool. We also include complete documentation and source code where avalible..

pwnDroid is writen in JavaScript and compiled to APK format using the free DroidScript IDE android app.

Features

pwnDroid: "Its like a version of Bugtroid thats actually worth downloading!.."

Get Involved

You want to contribute to pwnDroid? Awesome! You're more than welcome and you have a few options to choose from.


1.) You can visit our Github Repository for details on code submission.


2.) If you have developed an android app you want to add to the pwnDroid collection or you found an awesome open source app that we missed, just email us and let us know about it.


3.) You can donate Crypto Coins! ;-)

Developer Support

All of our software (including pwnDroid) is FREE to download and is not monitized in any way! If you want to support further development donating some Crypto Coin (of any flavor) is always welcome.
